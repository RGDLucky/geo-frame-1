# Frontend

React + Tailwind frontend with geospatial visualization.

## Quick Start

```bash
npm install
npm run dev
```

## Development

```bash
npm run dev     # Start dev server
npm run build  # Build for production
npm run preview # Preview production build
```

## Configuration

Configure API endpoint in your proxy setup or environment.

## Tech Stack

- React 18
- Tailwind CSS
- react-simple-maps (geospatial)
- d3-scale, d3-dsv (data processing)
- Vite (build tool)

## Features

- Interactive world map
- Geolocation markers
- Click-to-select markers
- Dark theme UI
